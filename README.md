# DARK-STORE-ANALYTICS-PLATFORM-
Darkstore Analytics Platform is an information-driven system that aims to improve warehouse operations using machine learning techniques. It has various functions, such as demand forecasting, route optimization, anomaly detection, and product affinity analysis, along with a dashboard and chatbot.
## Features

- Demand Forecasting for predicting future product demand
- Route Optimization for efficient delivery planning
- Anomaly Detection to identify unusual operational patterns
- Product Affinity Analysis to find frequently purchased product combinations
- Interactive Dashboard for data visualization
- Chatbot interface for querying analytics insights
